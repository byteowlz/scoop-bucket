# Byteowlz Scoop Bucket

A [Scoop](https://scoop.sh) bucket for Byteowlz tools.

## Installation

```powershell
scoop bucket add byteowlz https://github.com/byteowlz/scoop-bucket
```

## Available Apps

| App | Description |
|-----|-------------|
| mmry | A lean, local-first memory management system |

## Usage

```powershell
scoop install byteowlz/mmry
```

## Updating

```powershell
scoop update mmry
```
